To activate the theme, add or replace your current theme settings with the code 
below.


Settings for Afterglow
{
    "color_scheme": "Packages/Theme - Bamboo/Bamboo.tmTheme",
    "theme": "Bamboo.sublime-theme",
}

MAKE SURE TO RESTART SUBLIME AFTER ACTIVATING THE THEME
